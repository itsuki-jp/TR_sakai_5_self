ブラウザとは

ニコニコ動画 や Google などのホームページを表示するのに使うのが ブラウザ です。

設定の確認

ユーザ名が半角英数字だけになっているかどうか

Windows をお使いの方はご自身のユーザ名を確認する必要があります。
なぜなら、これから使用する多くのソフトウェアは海外で作られていて、日本語の入力を想定していない場合があるためです。特に、ユーザ名が英語（半角英数字）になっていないと正常に動作しないものがあります。

まず、スタートボタンを右クリックして、現れたメニューからファイル名を指定して実行をクリックします。

現れたウィンドウに以下の文字列を入力し、OKをクリックしましょう。コピー&ペーストで構いません。

cmd
ファイル名を指定して実行

OK をクリックすると、次のような黒い画面が表示されます。

ユーザ名

赤い枠で囲んだ部分の文字列が半角英数字だけの場合は問題ありません。
もし、日本語など半角英数字以外になっている場合は Windows のユーザを新しく作成する必要があります。

様々な種類のブラウザがありますが、このコースでは Google Chrome（グーグル・クローム）を利用して説明をしていきますので、 Google Chrome のインストール方法を紹介します。

JavaScript で計算をさせてみよう

それでは実際に、プログラミングの第一歩を踏み出してみましょう。ブラウザに計算をさせる命令を出してみます。

この console に、半角・英数字入力で
alert(25 + 25);
と入力して、 Enter を押してみましょう。

Windows の場合は 半角/全角 を押し、Mac の場合は 英数 を押すことで、半角・英数字入力ができます。

( は、 Shift を押しっぱなしにしながら 8 を押すと出すことができます。
) は、 Shift を押しっぱなしにしながら 9 を押すと出すことができます。
+ は、 Shift を押しっぱなしにしながら ; を押すと出すことができます。

Shift は、 PC によっては Shift と書かれておらず上矢印（⇧）のみで示されていることもありますが、いずれの場合も Z の左隣にあります。
なお、上に示した入力方法は一般的な日本語キーボードの場合です。英語キーボードなどでは、入力方法が異なりますのでご注意ください。

もし console に何も入力できないという場合は、console の > と表示されている部分のあたりをクリックしてから入力してみてください。

alert(表示したい計算式); と書くと計算ができます

練習

  1. 掛け算 8327 × 9874 の結果をアラートダイアログに表示させてみましょう。掛け算には * という記号を使います。 この記号のことを、アスタリスク と呼びます。 キーボードの Shift + : を押すと使うことができます。
    例: 2 * 3
  2. アラートダイアログに「こんにちは」という文章を表示してみましょう。文章は、'という記号で囲むとプログラムに記述できます。 この記号のことを、シングルクォーテーションと呼びます。 キーボードの Shift + 7 を押すと使うことができます。
    例: 'あいうえお'

【注意】記号は半角で入力しましょう。

記号の入力方法

一般に Shift を押しながら入力することで、キーボードの上部に印字された記号を入力できます。

半角と全角の切り替えは
Widows の場合： ［半角/全角］ を押す
Mac の場合： 英数 で半角、 かな で全角入力に切り替えることができます。
